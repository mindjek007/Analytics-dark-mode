# Privacy Policy — Analytics Dark Mode

**Last updated:** July 15, 2026

## Data Collection

**Analytics Dark Mode does not collect, store, transmit, or share any user data.**

The extension operates entirely locally within your browser:

- No personal information is collected
- No browsing data is collected
- No analytics or usage statistics are tracked
- No data is sent to any external server

## Permissions

The extension requests the following permissions solely for functionality:

| Permission | Purpose |
|------------|---------|
| `https://analytics.google.com/` | To apply the dark theme CSS only on Google Analytics pages |
| `storage` | To save your selected color scheme preference locally in your browser |

No data accessed through these permissions is ever collected, stored externally, or transmitted.

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.

## Changes

If this policy changes, users will be notified via the Chrome Web Store listing page.

## Contact

For questions about this privacy policy, please open an issue on the GitHub repository.
